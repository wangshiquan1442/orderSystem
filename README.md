# orderMealSystem

点餐系统

## 项目介绍
包含服务器后端，后台管理页面和用户点餐页面。用户通过扫描餐桌上的二维码打开点餐页面，完成点餐、支付操作，后台管理界面显示新订单、处理订单记录等。后端采用 springboot、 mybatis、 mysql、websocket、消息队列等；前端采用 vue、 vant ui、 element ui等框架技术.
